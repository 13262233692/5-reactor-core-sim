# 5-reactor-core-sim

核电站堆芯仿真培训游戏 - 基于 Godot 4 + C# 的二维严肃拟真游戏

## 项目概述

本项目是一款面向核电站操纵员培训的二维严肃拟真游戏，使用 C# 语言结合 Godot 4 引擎开发。系统实现了高精度的点堆中子动力学求解、热工水力计算、以及可视化的堆芯温度伪彩图渲染。

## 核心功能

### 1. 物理计算层
- **点堆中子动力学求解器**：使用四阶龙格-库塔（RK4）方法高精度迭代标准点堆动力学方程组
  - 6 组缓发中子先驱核
  - 中子代时间、缓发中子份额等参数可配置
  - 实时计算中子密度、先驱核浓度、堆功率

- **热工水力计算模块**：
  - 冷却剂温度计算（入口/出口/平均温度）
  - 包壳表面温度、燃料芯块中心温度计算
  - 临界热流密度（CHF）与偏离泡核沸腾比（DNBR）计算
  - 压降计算

- **信号滤波系统**：
  - 一阶低通滤波器
  - 移动平均滤波器
  - 一维卡尔曼滤波器
  - 中值滤波器
  - 级联滤波器（多种滤波器组合）
  - 流量噪声发生器（模拟真实流量波动）

### 2. 数据模型层
- **燃料组件模型**：157 个燃料组件的压水堆堆芯布局
  - 圆形对称布置（17x17 栅格，圆形边界）
  - 控制棒组件位置
  - 功率分布、温度分布、DNBR 分布

- **仿真快照**：线程安全的状态快照
- **控制命令**：类型安全的控制指令系统

### 3. 系统架构层
- **消息总线**：线程安全的发布-订阅模式
  - 命令队列（控制指令传入）
  - 状态快照（物理状态传出）
  - 锁机制保证线程安全

- **物理引擎**：独立后台线程运行
  - 100Hz 物理更新频率
  - 固定时间步长积分
  - 子步长细化计算
  - 仿真速度可调（0.1x ~ 50x）

- **输入系统**：键盘控制
  - 控制棒组操作（Q/W/E/R 提升，A/S/D/F 下插）
  - 紧急停堆（空格键）
  - 冷却剂流量调节（+/-）
  - 入口温度调节（I/K）
  - 仿真速度切换（1/2/3/4）
  - 显示模式切换（V/B/H）

### 4. 渲染层
- **堆芯横截面渲染**：
  - 157 个燃料组件的温度伪彩图
  - 支持温度模式和 DNBR 模式切换
  - 4 种配色方案（Jet、Thermal、Inferno、Rainbow）
  - 控制棒位置叠加显示

- **自定义着色器**：
  - thermal_palette.gdshader - 热伪彩着色器
  - 支持多种色图映射
  - 亮度、伽马调节

### 5. UI 界面层
- **参数面板**：
  - 堆芯参数（时间、功率、反应性、倍增时间）
  - 热工水力参数（流量、温度、压力）
  - 安全参数（最小 DNBR）
  - 控制棒位置指示

- **DNBR 警示系统**：
  - 两级警告（警告/危险）
  - 屏幕闪烁警示效果
  - DNBR 色标图例

- **帮助面板**：操作指南显示

## 操作说明

| 按键 | 功能 |
|------|------|
| Q / W / E / R | 提升控制棒组 A/B/C/D |
| A / S / D / F | 下插控制棒组 A/B/C/D |
| 空格 | 紧急停堆 (SCRAM) |
| + / - | 增加/减少冷却剂流量 |
| I / K | 提高/降低入口温度 |
| 1 / 2 / 3 / 4 | 仿真速度 0.1x / 1x / 10x / 50x |
| V | 切换配色方案 |
| B | 切换温度/DNBR 显示模式 |
| H | 显示/隐藏帮助面板 |
| Ctrl + R | 重置仿真 |

## 关键技术点

### 线程安全设计
- 使用 `lock` 机制保护共享数据
- 消息总线模式解耦物理线程与渲染线程
- 状态快照实现无锁读取

### 数值稳定性
- RK4 四阶龙格-库塔积分保证精度
- 子步长积分处理刚性方程
- 数值范围保护防止溢出

### 滤波算法
- 级联滤波（低通 + 移动平均）有效抑制流量噪声
- 保留有效信号的同时剔除数值毛刺
- 确保 DNBR 视觉警示清晰稳定

## 技术栈

- **引擎**: Godot 4.3
- **语言**: C# 12 / .NET 8
- **数值方法**: 四阶龙格-库塔 (RK4)
- **架构模式**: 消息总线、发布-订阅、生产者-消费者
